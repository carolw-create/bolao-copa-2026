🆕 ATUALIZAÇÕES PARA AS PRÓXIMAS FASES

❓ COMO FUNCIONA A PONTUAÇÃO

  • ✅Certo: você acertou o vencedor, mas errou os gols dos dois times.
  • ✅+ Gol: você acertou o vencedor E os gols exatos de um dos times.
  • 🎯Exato: você acertou o placar exato.
  • ❌0 pts: você errou o vencedor. Zero pontos.

🏆 MATA-MATA — COMO FUNCIONA

  • ⏱️Você aposta no placar após 120 minutos, antes de pênaltis.
  • 🔘Se o placar tiver vencedor, indique se o jogo foi para prorrogação ou terminou no tempo normal.
  • 🥅Se o placar for empate, o app entende automaticamente que foi para prorrogação e você escolhe o vencedor nos pênaltis.
  • 🎯+2 se acertar que o jogo foi para prorrogação (mas não pênaltis).
  • 🎯+4 se acertar que o jogo foi para pênaltis (inclui acertar o empate no placar).
  • 🎯+2 se acertar o vencedor nos pênaltis (além dos +4).
